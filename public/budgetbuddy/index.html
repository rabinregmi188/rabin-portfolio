About BudgetBuddy

BudgetBuddy is a personal finance dashboard built to make monthly money tracking easier, clearer, and less stressful. Instead of just storing transactions, it helps users understand where money is going and how each category compares to the budget they planned.

The app focuses on simple workflows: add income and expenses, assign categories, set monthly limits, and immediately see how spending patterns change over time. Everything is saved locally in the browser, so the experience feels fast and lightweight without needing a backend.

I designed this project as a polished front-end product demo that combines clean UI, practical analytics, and responsive layout design in a way that feels like a real personal finance tool rather than a classroom prototype.
